# Daily Motivation Dashboard

A React app that shows random motivational quotes with a like system.

## 🚀 Features
- Fetch random quotes
- Like/unlike quotes
- View liked quotes
- Persistent storage using localStorage

## 🛠 Tech Stack
- React
- Vite
- Tailwind CSS
- React Router
